Guia prático de seletores CSS

Demonstrações visuais para compreensão inicial

1. Entendendo o conceito

No CSS, os seletores determinam quais elementos da página serão estilizados.

Eles funcionam como um filtro que define onde cada regra será aplicada. 

p {
  color: green;
}

2. Seletor universal e por tipo

O seletor por tipo atinge todos os elementos de uma mesma tag.

Já o seletor universal aplica regras em todos os elementos da página. 

* {
  box-sizing: border-box;
}

p {
  color: lightgreen;
}

3. Classe e ID

Classes podem ser reutilizadas em diferentes elementos.

Isso ajuda a manter um padrão visual consistente. 

.destaque {
  background: #166534;
  padding: 10px;
}

IDs são únicos e identificam elementos específicos.

Devem ser usados apenas uma vez por página. 

#aviso {
  color: #86efac;
}

ul.menu-filho > li {
  background: #14532d;
}

5. Pseudo-classes

Passe o mouse aqui

Pseudo-classes representam estados de interação do usuário. 

.botao:hover {
  background: #22c55e;
}

input:focus {
  outline: 2px solid #4ade80;
}

Algumas pseudo-classes atuam na posição dos elementos. 

.lista-zebrada li:nth-child(2n) {
  background: #166534;
}

6. Pseudo-elementos

Texto com conteúdo visual adicional

Permitem inserir elementos visuais sem alterar o HTML. 

.pseudo::before {
  content: ">> ";
}

.pseudo::after {
  content: " <<";
}

Este texto destaca a primeira letra. 

.primeira-letra::first-letter {
  font-size: 2rem;
  color: #4ade80;
}

7. Especificidade

Este elemento recebe estilos de vários seletores

Quando há conflito entre regras, o CSS aplica a mais específica.

Normalmente, IDs têm maior prioridade, seguidos por classes e depois tipos. 

p { color: gray; }

.destaque { color: green; }

#principal { color: lightgreen; }

8. Resumo

  • Seletores definem onde o estilo será aplicado
  • Classes podem ser reutilizadas
  • IDs devem ser únicos
  • Combinadores usam a estrutura do HTML
  • Pseudo-classes representam estados
  • Pseudo-elementos criam efeitos visuais
  • Especificidade define prioridade
